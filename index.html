Mathematics in Biology — Fig. 1: Leaf Pattern

This graph visualizes the polar equation r(θ) = sin(θ) × cos(θ), rendered in Cartesian coordinates. It produces a leaf-like lobed shape, commonly found in plant morphology. Each point is generated based on angle θ, using x = r × cos(θ) and y = r × sin(θ). This demonstrates how polar functions can model natural symmetry in leaves and petals.

Mathematics in Biology — Fig. 2: DNA Helix

This graph shows a 3D mathematical helix, using the parametric equations: x = sin(t), y = cos(t), z = 0.05 × t. As the parameter t increases, the points trace a spiral in 3D space — mimicking the structure of a DNA strand. This model represents a single helix; a future extension can add a second strand for full biological accuracy.
